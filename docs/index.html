🚀 Quick Start

Up and running in under 2 minutes

# 1. Clone the repository
git clone https://github.com/systemslibrarian/AI-Conversation-Platform-The-Future-of-Multi-Agent-Collaboration.git
cd AI-Conversation-Platform-The-Future-of-Multi-Agent-Collaboration

# 2. Install dependencies
curl -LsSf https://astral.sh/uv/install.sh | sh
uv sync --all-extras

# 3. Configure API keys (need at least 2 providers)
cp .env.example .env && nano .env

# 4. Run a conversation
uv run aic-start --agent1 claude --agent2 chatgpt --topic "AI ethics" --turns 10 --yes

# 5. Or launch the interactive web demo
cd web && python demo.py
# Open http://localhost:5000
